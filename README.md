# waCSS

## Core idea
A minimalist, design system inspired primarily by Japanese WabiSabi and by Western Editorial Minimalism. The "wa" in the name comes from the Japanese concept of **wa (和)**, a concept translating to something like "harmony," "peace," or "balance", and also conformity within a social group. This project interprets harmony and conformity as structural inspiration. The css library should conform to globally recognized standards for accessibility, and be pleasant for users to use, and frictionless for developers. It prefers element styling over class styling, and encourages users to use actual semantic html, rather than inventing classes that do the same thing. Javascript is sometimes useful for accessibility behavior.

## Principles

### Harmony (Wa 和)
In harmony with users, developers, and international standards. Use the system. Prefer classless when possible. Accessible. Easy to learn.

### Simplicity (Kanso 簡素)
Clear away anything thats unnecessary. In development use intelligent CSS which reduces the lines of code. In appearance remove what is not needed.

### Spacing (Ma 間)
Emptiness, spacing, interval, timing. Use generous whitespace and slightly slower animations which create unhurried atmosphere.

### Asymmetry (Fukinsei 不均斉)
Asymmetry, balance achieved without mirror-symmetry, reflecting how nature is never perfectly uniform. Symmetry is not the only form of balance in design.

- Intrinsic sizing.** Elements size to their content first, then bend
   within relative limits. Layout adapts to content, never the reverse.
- **Clarity over decoration.** Every element must justify its presence —
   or it's removed.
- **Type is the interface.** Hierarchy comes from size, weight, and space —
   not decoration. Any typeface that serves the design is acceptable.
- **Whitespace is active.** Emptiness (*ma*, 間) is structural, not leftover
   space.
- **Restraint in color.** A neutral base, one sparing accent, used only to
   communicate — never to impress.
8. **Objectivity.** Neutral, legible, honest — no ornament, no skeuomorphism.
9. **Tasteful motion.** Smooth and communicative (an opening accordion,
   a sliding tab underline) — never decorative.
10. **Semantic HTML.** Real elements for real meaning; structure carries
    accessibility, styling hooks onto it.
- **Flatness.** Separation comes from line and space, not elevation. Shadows
   are almost never used.



On the page — the text column sits off-centre, weighted left; numerals and glyphs align to their own axis, not to the middle of the page. The equilibrium is real, but it is earned rather than measured. Nothing is mechanically halved.


### Shizen (Naturalness 自然)
Naturalness. Natural forms, soft borders, tasteful animations that feel natural, some biophilia.

### Kokō (Weathered Grace 考古)
Kokō is austerity and weathered grace. Soft, gentle colors. Non flashy animations. 


### Yūgen (Ethereal Beauty 幽玄)
Beauty that is felt more seen. Works by suggestion rather than statement. Everything should be understated. Leaving something withheld for the imagination to complete. Mujis "no brand" idea, that doesnt draw attention, and has quiet confidence. Detail that surfaces only on attention, such as swatch names that appear on hover.

### Datsuzoku (Freedom from convention 脱俗)
Breaking away from rule, habit, and formula that brings a liberating surprise. Find ways to be playful. Spontaneous feeling, whilst maintaining grace.

### Shibui (Restrained Depth 渋い)
Shibui is understated, unobtrusive beauty. 


### Seijaku (Energized calm  静寂)
Active, energized calm, the stillness at the centre rather than the absence of life. Nothing moves without cause. When harmony, simplicity, interval, and restraint hold together, what remains is a quiet the reader can rest in.
